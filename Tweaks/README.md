## 🚀 Quick Start: Tweaks & Debloat (`/Tweaks`)

Navigate to the `Tweaks/` folder, right-click **`Run-All-Tweaks.bat`**, and select **Run as Administrator**.

### Included Tweaks:
* **`Debloat-Apps.ps1`**: Removes native pre-installed bloatware (Xbox, Solitaire, Bing apps, etc.).
* **`Disable-ConsumerFeatures.ps1`**: Disables auto-installed sponsored apps and lock screen ads.
* **`Disable-SearchBloat.ps1`**: Disables Cortana and removes Bing web results from Start Menu searches.
* **`Disable-TelemetryServices.ps1`**: Disables the `DiagTrack` service and clears diagnostic logs.
* **`Enable-StorageSense.ps1`**: Configures Windows Storage Sense to run automated weekly cleanup.
